# spotify-system-analysis
A system analysis project based on a music streaming platform (Spotify-like system). The project focuses on analyzing existing system features and proposing enhancements by adding new functional requirements to improve usability and system performance.
# Features
- System requirement analysis  
- Identification of new functional requirements  
- System modeling using UML diagrams  
- Application of software engineering design principles

 # Techiques Used
 - UML Diagrams  
- Requirements Analysis  
- Software Design Concepts
  
# Purpose
This project was completed as part of a university course to practice system analysis and design techniques used in real software engineering projects.
